What's this website about?

The Daily Punch website was created with an intention to increase people's motivation by sharing their daily accomplishments with each other.

Register & Login

In order to post (punch in) daily accomplishments, user needs to register and log into the website. User can choose to either register by email address or via google account. Once logined in, user will be directed to the accomplishment page, where they can view all the submitted accomplishments on the current day (only the current day's accomplishments will be shown).

Submission

To submit daily accomplishments, click on the [Punch In] button on the accomplishment page. Only the latest submission for the day will be kept. Thus if user submits multiple times, only the latest one will be shown on the accomplishment page. User can submit and override the previous submission as many times as they want.
